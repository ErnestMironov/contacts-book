# contacts-book
---
Результат моего первого знакомства с фреймворком Vue.JS  
В проекте помимо самого Vue использованы Vuex и VueRouter

## Установка проекта
```
npm install
```

### Запуск режима разработки
```
npm run serve
```

### Сборка минифицированной версии
```
npm run build
```
